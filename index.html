 jumper.js

jumper.js is a javascript quick navigating tool that shows up in front of the web application binding it with a key combination.

Very useful to shortcut places in medium to large web applications, made for less clicking and faster navigation.

Try it now, press
Ctrl + J
or
⌘ command + J
or
◆ + J
or
Open Now

Installation

npm install --save @qoxcorp/jumper-js

or

<!-- For version 0.0.4 --> 
<script src="//cdn.rawgit.com/QOXCorp/jumper.js/47d26a631f07dbf6970ec92899a13a8acf7eaafe/dist/jumper.min.js"></script>

Requirements

The current version of Jumper.js requires Bootstrap 3, Bootstrap-dialog, Selectize.js and jQuery.
Non-bootstrap version is planned for future.

Optional

Include Font Awesome to show the spinning icon (when loading ajax data).

Jumper Options

Jumper({
    'data': [
        {
            name: 'Home',            
            help: 'Go to homepage.',
            action: {
                url: '/'
            }
        }
    ]
});
Name Description
debug Not required.
Type: Boolean.
Default: false.
Set to true to see some useful messages on the JS console.
bindFnKeys Not required.
Type: String.
Default: event.ctrlKey || event.metaKey.
Set the function key to bind.
  • event.ctrlKey
  • event.metaKey
  • event.altKey
You can combine using && or ||.
bindLetter Not required.
Type: Char.
Default: J.
Set the letter combination to use. (Bound function key plus bound letter triggers Jumper.Open();).
data Not required if using ajax & ajaxserver.
Type: Object Array.
Object array and sets the navigation options, each one has the following options:
{
    name: 'Link Name',        
    help: 'Some help for the user',
    keywords: 'some,keywords,separated,by,comma',
    navcode: 'Stablishes a shortcut code, this is not visible in the input',
    action: {
        url: 'When click or enter, Jumper go to it', // optional
        useItemHandler: false, // if true, this will execute the default item handler passing the 'item' object that contains all the object data, false by default
        method: (item) => {
            //execute this when selected, item object contains all the object data, not defined by default
            console.log(item);
        }
    },
    extra: {
        data1: 'my data' //you can set everything you want in this property, then you can use it from the default handler or the defined 'action.method'.
    }
}
Only name, help and action keys are required. All action parameters are optional, but you must define one of them otherwise nothing will happen at selection.
ajax Not required if using data.
Type: Boolean.
Default: false.
Set to true to enable remote data capabilities.
ajaxserver Required if using ajax: true.
Type: String.
Note: "action.method" is not supported as an ajax response.
Set the url of the data provider, the response must be in json format like the following example:
{ "response": [
        {
            "name": "Link Name",                
            "help": "Some help for the user",
            "keywords": "some,keywords,separated,by,comma",
            "navcode": "Stablishes a shortcut code, this is not visible in the input",
            "action": {
                "url": "/link-name"
            }
        }
    ]
}
cache Optional if using ajax: true.
Type: Boolean.
Default: false.
Set to true to enable data pre-caching when page loads.

Jumper Methods

Name Description
Jumper(options); Constructor, sets the options.
Jumper.Open(); Opens the Jumper modal without using the combination. Jumper must be initalized first using the constructor.
Jumper.ItemHandler(item); This is the handler when the item has "action.useItemHandler" set to true. You must override this function in order to use it. The item argument contains all of the item original data.

Localization

You can set the text strings of Jumper.js to match your local language (or website language). Just have to set up the following string before Jumper.js is initialized. 

// Spanish Language Texts for Jumper (Example).
    Jumper.TEXT_DefaultTitle = 'Navegación rápida';
    Jumper.TEXT_Loading = 'Cargando datos, espere porfavor...'; // For the Ajax Loading view.
    Jumper.TEXT_Navigating = 'Navegando, espere porfavor...';
    Jumper.TEXT_QuickNav = 'Escriba aquí un término para navegar rápidamente por el sitio web.';
    Jumper(options);
